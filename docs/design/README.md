## Adding New RFCs

**1. Name it**: `feature-name-YYYY-MM.md` (e.g., `multi-provider-support-2025-11.md`)

**2. Write it**: Problem → Motivation → Proposal → Alternatives

**3. Get feedback**: Create PR, iterate, merge when consensus reached

**4. Record decision**: Create ADR in `docs/decisions/` after implementation

See [docs/README.md](../README.md) for document lifecycle.
